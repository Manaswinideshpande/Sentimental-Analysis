Sentiment Analysis of Real-time Flipkart Product Reviews

Project Overview
This project focuses on building a Machine Learning model to classify customer reviews as positive or negative. It also analyzes negative reviews to identify key customer pain points, helping businesses improve their products and services.

Aim
To develop a sentiment analysis system that automatically classifies customer feedback and extracts meaningful insights from negative reviews to enhance customer satisfaction.

Features
Classifies reviews into Positive and Negative
Text preprocessing (removal of stopwords, punctuation, etc.)
Model training and evaluation
Identification of common issues in negative reviews
Data visualization of results

Technologies Used
Python
Machine Learning
Natural Language Processing (NLP)
Scikit-learn
Pandas, NumPy
Matplotlib / Seaborn

Workflow
Data Collection
Data Cleaning & Preprocessing
Text Vectorization (TF-IDF / CountVectorizer)
Model Training (e.g., Logistic Regression / Naive Bayes)
Model Evaluation
Insights Generation

Model Performance
F1 Score on Test Data: 0.95

Key Insights
Customers frequently complain about product quality, delivery delays, and customer service
Positive reviews highlight ease of use and value for money
Helps businesses identify areas for improvement
